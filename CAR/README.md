# 🚗 AI Car Price Predictor

AI Car Price Predictor is a modern machine learning web application that estimates the resale value of used cars based on key vehicle attributes. The application uses a trained regression model to analyze factors such as manufacturing year, present market price, kilometers driven, ownership history, fuel type, seller type, and transmission.

Designed with a premium glassmorphism interface, the dashboard provides an intuitive and interactive user experience with real-time predictions, confidence visualization, and AI-powered vehicle insights.

### Key Features

* 🚗 Accurate car resale price prediction using Machine Learning
* ✨ Modern Glassmorphism UI with responsive design
* 📊 Interactive Plotly Gauge Charts
* 💰 Real-time vehicle valuation
* 🤖 AI-powered insights and recommendations
* 📱 Mobile-friendly dashboard
* ⚡ Fast and lightweight Streamlit application

### Technologies Used

* Python
* Streamlit
* Scikit-Learn
* Pandas
* NumPy
* Plotly
* Joblib

### Use Case

This application helps car owners, buyers, dealers, and automotive businesses quickly estimate the market value of a vehicle before making buying or selling decisions.

### Model Inputs

* Manufacturing Year
* Present Price
* Kilometers Driven
* Number of Owners
* Fuel Type
* Seller Type
* Transmission Type

### Output

The system predicts the estimated resale value of the vehicle along with market confidence indicators and intelligent valuation insights.
